## Threat Scout
Full feature malware scanner. (in progress)

## ARP Scanner
A Python-based ARP scanner for discovering devices on a local network.  
Performs Layer 2 ARP requests to identify active hosts and retrieve IP addresses, MAC addresses, hostnames, and vendor information using an IEEE OUI database. Results are printed to the console and logged to a file.

## Port Scanner
A multi-threaded Python TCP port scanner built for learning and experimentation.  
Scans a target host for open ports, prioritizing clarity over speed, and provides basic timing and status output.

## Metadata Stripper
A command-line Python utility for removing metadata from image files.  
Strips EXIF data from images to reduce unintentional information disclosure while preserving visual content.

## Stegasaurus
A Python steganography tool for hiding encrypted messages inside image files.  
Uses least-significant-bit (LSB) techniques to embed and extract password-protected messages without visibly altering the image.
